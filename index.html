Using Filtrr

Designer friendly. Only the most basic JS knowledge needed.

  1. Step 1

    Download Filtrr Or fork filtrr.js on Github

  2. Step 2

    Reference the filtrr.js file in your html document

     <script type="text/javascript" src="filtrr.js"></script>

  3. Step 3

    Call the .filtrr() function on the relevant image

     function({
    $("#loader").filtrr(100);
  });

  4. Step 4

    Launch your website/app, then drop me a line to let me know how filtrr.js worked for you.
